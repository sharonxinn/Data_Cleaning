# Data_Cleaning
This project involved cleaning, validating, and preparing large datasets using SQL to ensure data accuracy, consistency, and usability for analytics and reporting. The focus was on identifying and correcting errors, handling missing values, standardizing formats, and eliminating duplicates, enabling reliable insights from raw data.

# Key Contributions:
-Performed data profiling to understand the quality and structure of raw datasets.
-Used SQL queries to identify and remove duplicate records and inconsistent entries.
-Applied data type conversions and format standardization to ensure uniformity across tables.
-Handled missing values through conditional updates, default values, or record removal.
-Joined and merged multiple tables to create a clean, consolidated dataset ready for analysis.
-Implemented validation checks to maintain data integrity post-cleaning.

# Skills Gained:
-Advanced SQL querying (SELECT, JOIN, CASE, COALESCE, GROUP BY)
-Data validation, error detection, and correction
-Handling nulls, duplicates, and inconsistent data
-Preparing datasets for business intelligence and analytics

# Tools Used:

MySQL 9.0
My SQL Workbench 8.0 CE

# Impact / Insights:
-Improved dataset reliability and accuracy for downstream analytics.
-Reduced data processing errors and ensured consistent reporting.
-Enhanced ability to extract actionable insights from clean and structured data.
